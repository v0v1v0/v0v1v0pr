rorq_flipper R Documentation

Rorqual flipper length

Description

Rorqual flipper length

Usage

rorq_flipper(species, length_m)

Arguments

species

a vector of species codes
length_m

a vector of lengths in meters

Value

a vector of flipper lengths in m

Examples

# A 22m blue whale
rorq_flipper("bw", 22)

# A 7m minke
rorq_flipper("ba", 7)
