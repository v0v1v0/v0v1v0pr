seq2 R Documentation

Sequence generation

Description

Generates a regular sequence from the minimum to the maximum of a vector.

Usage

seq2(x, int = 999)

Arguments

x

numeric vector.

int

number of values to be generated (int breaks).

Author(s)

Maxime HERVE <maxime.herve@univ-rennes1.fr>

See Also

seq

Examples

seq2(rnorm(30))
