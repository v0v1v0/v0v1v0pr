network_data R Documentation

Network data

Description

  • network_data() returns a data frame that can easily be converted to various network classes.

  • network_graph() returns a igraph object

Usage

network_data(x, e = c("mention", "retweet", "reply", "quote"))

network_graph(x, e = c("mention", "retweet", "reply", "quote"))

Arguments

x

Data frame returned by rtweet function
e

Type of edge/linkâ€“i.e., "mention", "retweet", "quote", "reply". This must be a character vector of length one or more. This value will be split on punctuation and space (so you can include multiple types in the same string separated by a comma or space). The values "all" and "semantic" are assumed to mean all edge types, which is equivalent to the default value of c("mention", "retweet", "reply", "quote")
