is_ggplot R Documentation

Reports whether plotstring encode a ggplot object

Description

Reports whether plotstring encode a ggplot object

Usage

is_ggplot(plotstring)

Arguments

plotstring

A character

Examples

require(ggplot2)
is_ggplot("plot(iris)")
is_ggplot("ggplot(iris,aes(x=Sepal.Length))+geom_histogram()")
