decodeQA R Documentation

Decode QA flags to bit-words

Description

Intended for use with Landsat 16-bit QA bands. Decodes pixel quality flags from integer to bit-words.

Usage

decodeQA(x)

Arguments

x

Integer (16bit)

Value

Returns the decoded QA values from an integer

See Also

encodeQA

Examples

decodeQA(53248)
