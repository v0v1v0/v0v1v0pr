Description

add alpha level to color that lacks one

Usage

AddAlpha(plotclr, alpha = 0.5, verbose = 0)

Arguments

plotclr

color to be modified
alpha

alpha level
verbose

level of verbosity

Value

modified color with alpha value

Author(s)

Markus Loecher
