tg.isPointTier R Documentation

tg.isPointTier

Description

Returns TRUE if the tier is PointTier, FALSE otherwise.

Usage

tg.isPointTier(tg, tierInd)

Arguments

tg

TextGrid object
tierInd

tier index or "name"

Value

TRUE / FALSE

See Also

tg.isIntervalTier, tg.getTierName, tg.findLabels

Examples

tg <- tg.sample()
tg.isPointTier(tg, 1)
tg.isPointTier(tg, "word")
