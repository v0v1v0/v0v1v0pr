Description

compute the price of a partition solution (and check is valid).

Usage

score_solution(x, solution)

Arguments

NumericMatix, for j>=i x(i,j) is the cost of partition element [i,...,j] (inclusive).
solution

vector of indices

Value

price

Examples


x <- matrix(c(1,1,5,1,1,0,5,0,1), nrow=3)
s <- c(1, 2, 4)
score_solution(x, s)
