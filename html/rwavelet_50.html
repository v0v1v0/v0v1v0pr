Description

Apply Soft Threshold

Usage

SoftThresh(y, t)

Arguments

y

Noisy Data.
t

Threshold.

Value

x filtered result (y 1_|y|>t).

See Also

HardThresh

Examples

f <- MakeSignal('HeaviSine', 2^3)
qmf <- MakeONFilter('Daubechies', 10)
L <- 0
wc <- FWT_PO(f, L, qmf)
thr <- 2
wct <- SoftThresh(wc, thr)
fsoft <- IWT_PO(wct, L, qmf)
