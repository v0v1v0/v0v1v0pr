gd_properties R Documentation

Get Properties of an Image Collection

Description

Get Properties of an Image Collection

Usage

gd_properties(x)

Arguments

x

geedim.collection.MaskedCollection object

Value

data.frame containing properties table from x; NULL if no properties table.

Examples



library(terra)

b <- terra::vect('POLYGON((-121.355 37.560, 
                           -121.355 37.555,
                           -121.350 37.555, 
                           -121.350 37.560,
                           -121.355 37.560))',
                 crs = "OGC:CRS84")

if (gd_is_initialized()) {
  x <- gd_search(gd_collection_from_name("USGS/3DEP/1m"),
                 region = gd_region(b))
  gd_properties(x)
}
