methods_rxylib R Documentation

methods_rxylib

Description

S3-methods support by the package rxylib. Listed functions can be passed directly into S3 generics (e.g., plot, print) without reshaping the data.

Usage

## S3 method for class 'rxylib'
print(x, ...)

## S3 method for class 'rxylib'
plot(x, block = NULL, ...)

Arguments

x

(required): input object
...

further arguments that can be passed to the method
block

numeric (with default): select block for plotting, e.g. c(1:2).
