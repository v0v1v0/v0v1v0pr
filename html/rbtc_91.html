stopbtc R Documentation

Stop bitcoind server process

Description

This function stops a running bitcoind process. It calls bitcoin-cli stop via the R function system().

Usage

stopbtc(confbtc)

Arguments

confbtc

CONRPC object, returned from conrpc().
