TRANmat R Documentation

Translation Matrix

Description

Create a 4 by 4 translation matrix

Usage

TRANmat(x, y, z)

Arguments

x

x-translation
y

y-translation
z

z-translation

Value

Matrix suitaqble for translating a 3D body.
