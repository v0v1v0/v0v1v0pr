Is an object the empty coding?

Description

Is an object the empty coding?

Usage

is_empty_coding(x)

Arguments

x

An object

Value

TRUE/FALSE if the object is identical to 'empty_coding()'

Examples

is_empty_coding(empty_coding())
is_empty_coding(coding())
is_empty_coding(coding(code("Yes", 1), code("No", 0)))
