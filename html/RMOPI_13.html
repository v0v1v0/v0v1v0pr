ggpacf R Documentation

Plot the Pacf Figure

Description

Plot the Pacf figure with observations of a single variable beautifully.

Usage

ggpacf(data, lag = 10)

Arguments

data

vector of observations
lag

the maximum lag to calculate the pacf

Value

A ggplot figure of the pacf

Examples

swan <- rGarch(len = 180)
ggpacf(swan^2, 20)
