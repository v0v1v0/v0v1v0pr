MirrorFilt R Documentation

Apply (-1)^t modulation

Description

h(t) = (-1)^(t-1) * x(t), 1 <= t <= length(x)

Usage

MirrorFilt(x)

Arguments

x

1-d signal.

Value

h 1-d signal with DC frequency content shifted to Nyquist frequency

See Also

DownDyadHi.

Examples

x <- MakeSignal('HeaviSine',2^3)
h <- MirrorFilt(x)
