getbestblockhash R Documentation

RPC-JSON API: getbestblockhash

Description

Returns the hash of the best (tip) block in the longest blockchain.

Usage

getbestblockhash(con)

Arguments

con

object of class CONRPC.

Value

A S4-object of class ANSRPC.

Author(s)
