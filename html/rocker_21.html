Information whether all records were retrieved.

Description

Information whether all records were retrieved.

Usage

hasCompleted(db, ...)

Arguments

db

rocker object
...

Optional, additional suitable parameters passed to DBI::dbHasCompleted()

Value

TRUE or FALSE
