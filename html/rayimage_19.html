make_vignette_overlay R Documentation

Make Vignette Overlay

Description

Makes an overlay to simulate vignetting in a camera

Usage

make_vignette_overlay(
  width,
  height,
  intensity = 0.3,
  radius_multiplier = 1.3,
  radius = NULL,
  color = "#000000"
)

Arguments

width

Width of the image.
height

Height of the image.
intensity

Default 0.4. 1 is max intensity, 0 is min.
radius_multiplier

Default 1.3.
radius

Default NULL. Max of height or width, divided by 2.
color

Default black.
