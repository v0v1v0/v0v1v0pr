list_gt3x R Documentation

List full paths to all gt3x files in a directory

Description

List full paths to all gt3x files in a directory

Usage

list_gt3x(path)

Arguments

path

Path(s) to file(s)

See Also

Other file manipulations: gt3x_datapath(), is_gt3x(), unzip.gt3x()

Examples

path <-
  system.file(
    "extdata",
    package = "read.gt3x")
list_gt3x(path)
## Not run: 
list_gt3x(gt3x_datapath())

## End(Not run)
