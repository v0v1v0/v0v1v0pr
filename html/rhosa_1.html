.blackman_window R Documentation

Blackman window function

Description

Calculate

\frac{42}{100} - \frac{\cos(2 \pi x)}{2} + \frac{8 \cos(4 \pi x)}{100}.

Usage

.blackman_window(x)

Arguments

x

A real number in [0, 1].

Value

A real number in [0, 1].

See Also

.hamming_window and .hann_window.
