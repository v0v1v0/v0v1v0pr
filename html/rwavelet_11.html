dyadlength R Documentation

Find length and dyadic length of array

Description

Find length and dyadic length of array

Usage

dyadlength(x)

Arguments

x

array of length n = 2^J (hopefully).

Value

n length(x).

J least power of two greater than n.

See Also

quadlength, dyad

Examples

x <- MakeSignal('Ramp', 8)
dyadlength(x)
