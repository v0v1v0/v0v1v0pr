isNum R Documentation

isNum

Description

Returns TRUE / FALSE whether it is exactly 1 number (numeric or integer vector of length 1, non-missing)

Usage

isNum(num)

Arguments

num

variable to be tested

Value

TRUE / FALSE

See Also

isInt, isLogical, isString

Examples

isNum(2)
isNum(2L)
isNum(-2)
isNum(-2L)
isNum(2.1)
isNum(-2.1)
isNum(1:5)
isNum(NA_real_)
isNum(numeric(0))
