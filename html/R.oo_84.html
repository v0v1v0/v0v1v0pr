check.Rdoc R Documentation

Checks the compiled Rd files

Description

Checks the compiled Rd files.

Usage

## S3 method for class 'Rdoc'
check(this, manPath=getManPath(this), verbose=FALSE, ...)

Arguments

manPath

The path to the Rd files (character string).
verbose

If TRUE, extra information is outputted.
...

Not used.

Details

Internally the tools package is used.

Value

Returns a printable object, which, if non-empty will show the errors.
