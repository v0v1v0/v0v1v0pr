similar_genres R Documentation

To return similar genres to a given genre

Description

To return similar genres to a given genre

Usage

similar_genres(api_key, genre = NA, description = T, urls = T,
  start = NA, results = 15)

Arguments

api_key

Echo Nest API key
genre

the genre name
description

genre's description
urls

genre's urls
start

the desired index of the first result returned
results

the number of results desired

Value

data frame giving similar genres

Examples

## Not run: 
data=similar_genres(api_key,genre="rock")

## End(Not run)
