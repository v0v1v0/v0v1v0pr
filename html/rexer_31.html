vector_to_string R Documentation

Transforms a vector of strings into a string

Description

Insert the separator ("<|>") to later facilitate the reverse operation.

Usage

vector_to_string(vector)

Arguments

vector

A vector of strings.

Value

A string.

Examples


s <- vector_to_string(c('Addition', '+'))
