tg.setTierName R Documentation

tg.setTierName

Description

Sets (changes) name of tier of the given index.

Usage

tg.setTierName(tg, tierInd, name)

Arguments

tg

TextGrid object
tierInd

tier index or "name"
name

new "name" of the tier

See Also

tg.getTierName

Examples

tg <- tg.sample()
tg2 <- tg.setTierName(tg, "word", "WORDTIER")
tg.getTierName(tg2, 4)
