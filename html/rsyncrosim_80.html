filepath R Documentation

Retrieves the path to a SyncroSim object on disk

Description

Retrieves the path to a SyncroSim Session, SsimLibrary, Project, Scenario, of Folder on disk.

Usage

filepath(ssimObject)

## S4 method for signature 'character'
filepath(ssimObject)

## S4 method for signature 'Session'
filepath(ssimObject)

## S4 method for signature 'SsimObject'
filepath(ssimObject)

## S4 method for signature 'Folder'
filepath(ssimObject)

Arguments

ssimObject

Session, Project, SsimLibrary, or Folder object

Value

A character string: the path to a SyncroSim object on disk.

Examples

## Not run: 
# Specify file path and name of new SsimLibrary
myLibraryName <- file.path(tempdir(), "testlib")

# Set up a SyncroSim Session and SsimLibrary
mySession <- session()
myLibrary <- ssimLibrary(name = myLibraryName, session = mySession)

# Get the file path
myFilePath <- filepath(myLibrary)

## End(Not run)
