SsimLibrary-class R Documentation

SyncroSim Library class

Description

SsimLibrary object representing a SyncroSim Library. A SsimLibrary is the highest level of organization in the SyncroSim workflow and contains at least one Project.

Slots

session

Session object

filepath

character string. The path to the SsimLibrary on disk

datasheetNames

character string. The name and scope of all Datasheets in the SsimLibrary.

See Also

See ssimLibrary for options when creating or loading a SyncroSim SsimLibrary.
