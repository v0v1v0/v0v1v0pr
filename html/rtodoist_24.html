init_project R Documentation

Init project

Description

Create a project with a specific to-do list for different people

Usage

init_project(
  project_id,
  tasks_list,
  try_again = 3,
  time_try_again = 3,
  verbose = TRUE,
  responsible = NULL,
  token = get_todoist_api_token()
)
