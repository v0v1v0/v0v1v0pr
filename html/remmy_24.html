lemmy_delete_private_message R Documentation

Delete a private message.

Description

HTTP.POST /private_message/delete

Usage

lemmy_delete_private_message(
  private_message_id,
  deleted,
  auth,
  .lemmy_instance = getOption("lemmy_instance", "https://lemmy.world")
)

Arguments

private_message_id

A numeric value.
deleted

A logical value.
auth

A character value.
.lemmy_instance

Default: "https://lemmy.world". Change the default instance by setting the lemmy_instance options().
