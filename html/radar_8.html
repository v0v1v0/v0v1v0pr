DopplerFmax R Documentation

Maximum frequency given PRF

Description

DopplerFmax returns the PRF for a maximum frequency From Rinehart (1997), Eqn 6.8

Usage

DopplerFmax(PRF)

Arguments

PRF

Pulse repetition frequency [Hz]

Value

f

Maximum frequency [Hz]

Author(s)

Jose Gama

Source
