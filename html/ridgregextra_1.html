ridgereg_k R Documentation

Ridge regression results with an automatically selected k value

Description

Ridge regression with a selected k value

Usage

ridgereg_k(x, y, a, b)

Arguments

x

Explanatory variables (Dataframe, matrix)
y

Dependent variables (Dataframe, vector)
a

Lower bound of k
b

Upper bound of k

Value

A list of lists

Examples

library("mctest")
x <- Hald[,-1]
y <- Hald[,1]
ridgereg_k(x,y,a=0,b=1)

library(isdals)
data(bodyfat)
x <- bodyfat[,-1]
y <- bodyfat[,1]
ridgereg_k(x,y,a=0,b=1)
