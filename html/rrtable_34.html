df2RcodeTable R Documentation

Make a flextable with a data.frame

Description

Make a flextable with a data.frame

Usage

df2RcodeTable(df, bordercolor = "gray", format = "pptx", eval = TRUE)

Arguments

df

A data.frame
bordercolor

A border color name
format

desired format. choices are "pptx" or "docx"
eval

logical. Whether or not evaluate the code

Value

A flextable object
