uuid_generate_nil R Documentation

Generates Nil UUIDs

Description

Function generates nil uuids.

Usage

uuid_generate_nil(n = 1L)

Arguments

n

Number of generated UUIDs.

Value

Character vector with UUIDs.

# generate nil UUIDs
