Gets the name

Description

Gets the name.

Usage

## S3 method for class 'FullNameInterface'
getName(this, ...)

Arguments

...

Additional arguments passed to *getFullName().

Details

The name is the part of the fullname that precedes any comma. For instance, the name of foo,a.2,b is foo.

Value

Returns a character.
