requireR R Documentation

Loads given files and modularizes the given function

Description

Loads given files and modularizes the given function

Usage

requireR(...)

Arguments

...

A set of filenames, and/or ending with the modularizable function.

Examples

## Not run: 
#hello.R
requireR(function() {
  "hello"
})

#world.R
requireR(function() {
  "world"
})

requireR(
  "hello.R",
   "world.R",
   function(hello, world) {
     paste(hello, world)
   })

## End(Not run)
