put_tiddler(

  text,

  tags = NULL,
  fields = NULL
)

title

tiddler title

tiddler text
type

tiddler type
tags

tiddler tags

a named vector for tiddler fields

Value

null if success

Examples

title <- "New tiddler"

            fields = fields)
