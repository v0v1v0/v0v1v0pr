update_all R Documentation

update_all

Description

update all packages

Usage

update_all(
  check_R = TRUE,
  which = c("CRAN", "BioC", "github"),
  lib.loc = NULL,
  ...
)

Arguments

check_R

whether check R version
which

repo (CRAN, BioC, github) to update
lib.loc

location of library, default is NULL and will set to .libPaths()
...

additional parameters to install packages

Author(s)

Guangchuang Yu

Examples

## Not run: 
library(rvcheck)
update_all()

## End(Not run)
