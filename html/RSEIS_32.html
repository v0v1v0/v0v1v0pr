COMPorder R Documentation

Seismic Component Order

Description

Set seismic component order

Usage

COMPorder(STNS, COMPS)

Arguments

STNS

stations

COMPS

components

Details

Sets up components so they are ordered according to V, N, E. used internally in swig.

Value

order vector
