getnetworkinfo R Documentation

RPC-JSON API: getnetworkinfo

Description

Returns an object containing various state info regarding P2P networking.

Usage

getnetworkinfo(con)

Arguments

con

object of class CONRPC.

Value

A S4-object of class ANSRPC.

Author(s)

Bernhard Pfaff
