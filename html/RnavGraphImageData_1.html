binaryalphadigits R Documentation

Binary Alphadigits

Description

Binary 20x16 digits of "0" through "9" and capital "A" through "Z". 39 examples of each class.

From Simon Lucas' (sml@essex.ac.uk), Algoval system.

Usage

binaryalphadigits

Format

Data frame with one image per row.
