Show Palette Photo

Description

This function shows photo that inspired a palette

Usage

ShowPalettePhoto(name = NULL, ...)

Arguments

name

palette name
...

Numeric, complex, or logical vectors

Value

image of reference photo with palette

Examples

ShowPalettePhoto("GoldenTemple")
ShowPalettePhoto("Kulfi")
ShowPalettePhoto("AmritsariPedeWaliLassi")
