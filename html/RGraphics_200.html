grid.utext R Documentation

Draw Underlined Text

Description

Draws a single piece of text with a line beneath.

Usage

grid.utext(label, x=.5, y=.5, ..., name="utext") 
textCorners(x)

Arguments

label

A character value.
x,y

Numeric or unit value.
...

Further arguments passed to grid.text.
name

Character value.
