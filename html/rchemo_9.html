checkna R Documentation

Find and count NA values in a dataset

Description

Find and count NA values in each row observation of a dataset.

Usage

checkna(X)

Arguments

X

A dataset.

Value

A data frame summarizing the numbers of NA by rows.

Examples


X <- data.frame(
  v1 = c(NA, rnorm(9)), 
  v2 = c(NA, rnorm(8), NA),
  v3 = c(NA, NA, NA, rnorm(7))
)
X

checkna(X)
