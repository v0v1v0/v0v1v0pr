Description

Rorqual fluke length

Usage

rorq_fluke(species, length_m)

Arguments

species

a vector of species codes
length_m

a vector of lengths in meters

Value

a vector of fluke lengths in m

Examples

# A 22m blue whale
rorq_fluke("bw", 22)

# A 7m minke
rorq_fluke("ba", 7)
