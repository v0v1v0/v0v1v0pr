public_daystats R Documentation

24 Hour Stats For A Given Product

Description

This function is a public function and will fetch get 24 hr stats for the provided currency pair (product_id). Volume is in base currency units. open, high, low are in quote currency units.

Usage

public_daystats(product_id = "BTC-USD")

Arguments

product_id

Optional character parameter. This is a case insensitive value of the product id for which the order book is desired. Default to 'BTC-USD'. For all valid product ids, refer to public_info.

Value

Dataframe with a single row of product's last 24 hour stats.

Examples

## Not run: 
public_daystats()

## End(Not run)
