CROSSL R Documentation

Vector Cross Product

Description

returns cross product of two vectors in list format

Usage

CROSSL(A1, A2)

Arguments

A1

list x,y,z
A2

list x,y,z

Value

List

x,y,z

input vector
az

azimuth, degrees
dip

dip, degrees

Author(s)

Jonathan M. Lees<jonathan.lees@unc.edu>

See Also

RSEIS::xprod

Examples


A1 = list(x=1,y=2, z=3)
A2 = list(x=12,y=-2, z=-5)

N = CROSSL(A1, A2)
