rgsp R Documentation

Repetitive Group Sampling Plan Based on Cpk

Description

The rgsp package provides functionalities to calculate Sample Number and Average Sample Number for a Repetitive Group Sampling Plan based on Cpk as given in Aslam et al. (2013).
