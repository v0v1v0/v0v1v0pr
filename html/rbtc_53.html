getmempoolinfo R Documentation

RPC-JSON API: getmempoolinfo

Description

Returns details on the active state of the TX memory pool.

Usage

getmempoolinfo(con)

Arguments

con

object of class CONRPC.

Value

A S4-object of class ANSRPC.
