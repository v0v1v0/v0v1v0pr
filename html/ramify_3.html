argmax R Documentation

Row/Column Max/Min Indices

Description

Returns the indices of the maximum or minimum values along an axis.

Usage

argmax(x, rows = TRUE)

argmin(x, rows = TRUE)

Arguments

x

A matrix.
rows

If TRUE (the default) the indices of each row max/min is returned.

Value

A vector of indices.

Examples

m <- mat("94, 20, 44; 40, 92, 51; 27, 69, 74")
argmax(m)
argmin(m)
