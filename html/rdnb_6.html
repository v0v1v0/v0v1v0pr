print_query R Documentation

Print search query

Description

print_query prints out the query used for a DNB-search request.

Usage

print_query(dnb_obj)

Arguments

dnb_obj

the DNB-serch object returned by dnb_search or dnb_advanced.

Value

Query string.

Examples

## Not run: 
dnb.srch <- dnb_search(title="katze")
print_query(dnb.srch)

## End(Not run)
