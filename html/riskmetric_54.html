memoise_cran_mirrors R Documentation

Fetch CRAN Mirrors Info

Description

Fetch CRAN Mirrors Info

Usage

memoise_cran_mirrors(all = TRUE, ..., .local = getOption("riskmetric.tests"))

Arguments

all

default TRUE, passed to utils[getCRANmirrors]
...

additional arguments passed to utils[getCRANmirrors]
.local

an optional local directory to source the CRAN package index from, defaulting to getOption("riskmetric.tests"), used for isolating repository requests during testing.

Value

a data frame with mirror information
