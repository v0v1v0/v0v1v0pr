is.tril R Documentation

Lower Triangular Matrix Test

Description

Determine if a Matrix is Lower Triangular

Usage

is.tril(x)

Arguments

x

A matrix

Value

Logical indicating whether the given matrix is lower triangular.

Examples

m <- mat("1, 0, 0, 0; -1, 1, 0, 0; -2, -2, 1, 0; -3, -3, -3, 1")
is.tril(m)
is.tril(eye(3, 5))
