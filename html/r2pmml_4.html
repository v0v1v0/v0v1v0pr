as.scorecard R Documentation

Converts a "glm" object to a "scorecard" object.

Description

Converts a "glm" object to a "scorecard" object.

Usage

as.scorecard(glm, odds = 10, base_points = 500, pdo = 100)

Arguments

glm

A "glm" object with binomial family link function.
odds

Odds ratio at base odds.
base_points

Points where odds ratio is defined.
pdo

Points to double the odds.

Value

A "scorecard" object.
