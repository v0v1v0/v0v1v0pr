getContribUrl.Package R Documentation

Gets the URL(s) from where this package can be installed

Description

Gets the URL(s) from where this package can be installed by first looking for comma or semicolon separated URLs at the optional ContribURL line in the DESCRIPTION file of the package. If no such line exists, getUrl() is used.

Usage

## S3 method for class 'Package'
getContribUrl(this, ...)

Arguments

...

Not used.

Value

Returns a vector of character strings.

Author(s)

Henrik Bengtsson

See Also

For more information see Package.

Examples

  pkg <- Package("R.oo")
  print(getContribUrl(pkg))
