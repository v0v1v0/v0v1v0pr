gt3x_download R Documentation

Download and unzip a zipped gt3x file

Description

Download and unzip a zipped gt3x file

Usage

gt3x_download(url, exdir, verbose = TRUE)

Arguments

url

url of the file to download
exdir

directory to extract the zip file
verbose

print diagnostic messages

Value

file path of exdir
