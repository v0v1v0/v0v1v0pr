get_gene_id R Documentation

Get all gene IDs in a DESeqDataSet for a given gene name.

Description

Get all gene IDs in a DESeqDataSet for a given gene name.

Usage

get_gene_id(gene_name, dds)

Arguments

gene_name

A gene name
dds

A DESeqDataSet

Value

A character vector

Examples

get_gene_id("HBA1", T47D)
