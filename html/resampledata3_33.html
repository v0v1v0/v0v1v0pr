Phillies2009 R Documentation

Philadelphia Phillies data 2009

Description

Baseball data for Philadelphia Phillies during the 2009 season.

Usage

Phillies2009

Format

A data frame with 162 observations on the following 8 variables.

Date

Date of game

Location

Game played where: Away, Home

Outcome

Outcome of game: Lose, Win

Outcome2

Outcome recoded: 1=win, 0 = lose

Hits

Number of hits

Doubles

Number of doubles

Homeruns

Number of homeruns

StrikeOuts

Number of strikeouts

Source

https://www.baseball-reference.com/
