get_items R Documentation

Get Dota2 Item List

Description

Get a list of in-game items

Usage

get_items()

Value

get_items returns a list of in-game items, which include the columns of name, cost, secret_shop, side_shop, recipe and localized name.

Examples

## Not run: 
itemList <- get_items()
head(itemList)

## End(Not run)
