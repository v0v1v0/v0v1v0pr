isCursorEmpty R Documentation

Check if cursor is empty

Description

Check whether a given cursor is fully drained and will output no more datum. The function never blocks; also verifies that the underlying connection is useful.

Usage

isCursorEmpty(cursor)

Arguments

cursor

Cursor to check; a result of r()$...$run(...).

Value

TRUE if cursor has no more data to return.

Note

It is possible that cursorNext will return NULL just after isCursorEmpty returns FALSE. Changefeeds cursors (made with r()$...$changes()$...) will never become empty (provided that connection won't become broken).
