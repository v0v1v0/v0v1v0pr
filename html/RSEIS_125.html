insertNAs R Documentation

Insert NA in a vector at given break points

Description

Insert NA in a vector at given break points

Usage

insertNAs(v, w)

Arguments

v

original vector

w

break points

Details

Used for plotting lines that wrap around.

Value

vector with NA inserted

Author(s)

Jonathan M. Lees<jonathan.lees@unc.edu>

Examples


a  <-  1:20
b  <-  insertNAs(a, c(5, 12))
b
