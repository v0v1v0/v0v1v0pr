tg.getNumberOfPoints R Documentation

tg.getNumberOfPoints

Description

Returns number of points in the given point tier.

Usage

tg.getNumberOfPoints(tg, tierInd)

Arguments

tg

TextGrid object
tierInd

tier index or "name"

Value

integer

See Also

tg.getNumberOfIntervals

Examples

tg <- tg.sample()
tg.getNumberOfPoints(tg, "phoneme")
