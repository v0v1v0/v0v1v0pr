Symbolic Median

Description

This function compute the median for symbolic intervals.

Usage

median(x, na.rm = FALSE, method = c("centers", "interval"), ...)

median(x, ...)

Arguments

x

A symbolic interval.
na.rm

As in R median function.
method

The method to be use.
...

As in R median function.
