FWT_TI R Documentation

Translation Invariant Forward Wavelet Transform

Description

1. qmf filter may be obtained from MakeONFilter. 2. usually, length(qmf) < 2^(L+1). 3. To reconstruct use IWT_TI.

Usage

FWT_TI(x, L, qmf)

Arguments

x

array of dyadic length n=2^J.
L

degree of coarsest scale.
qmf

orthonormal quadrature mirror filter.

Value

TIWT stationary wavelet transform table.

See Also

IWT_TI, MakeONFilter.

Examples

x <- MakeSignal('Ramp', 8)
L <- 0
qmf <- MakeONFilter('Haar')
TIWT <- FWT_TI(x, L, qmf)
