agAttributes R Documentation

Getting and setting antigen attributes

Description

These functions get and set the antigen attributes for a map.

Usage

agIDs(map)
agIDs(map) <- value
agDates(map)
agDates(map) <- value
agReference(map)
agReference(map) <- value
agNames(map)
agNames(map) <- value
agExtra(map)
agExtra(map) <- value
agPassage(map)
agPassage(map) <- value
agLineage(map)
agLineage(map) <- value
agReassortant(map)
agReassortant(map) <- value
agStrings(map)
agStrings(map) <- value
agContinent(map)
agContinent(map) <- value

Arguments

map

The acmap data object
value

New value to set

Value

Returns either the requested attribute when using a getter function or

the updated acmap object when using the setter function.
