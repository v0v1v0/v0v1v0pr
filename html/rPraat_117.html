tg.removeTier R Documentation

tg.removeTier

Description

Removes tier of the given index.

Usage

tg.removeTier(tg, tierInd)

Arguments

tg

TextGrid object
tierInd

tier index or "name"

Value

TextGrid object

See Also

tg.insertNewIntervalTier, tg.insertNewPointTier, tg.duplicateTier

Examples

## Not run: 
tg <- tg.sample()
tg.plot(tg)
tg2 <- tg.removeTier(tg, "word")
tg.plot(tg2)

## End(Not run)
