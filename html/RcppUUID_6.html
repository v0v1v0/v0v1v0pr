uuid_validate R Documentation

Validate UUIDs

Description

Function validates uuids.

Usage

uuid_validate(x)

Arguments

x

Character vector.

Value

Logical vector.

Examples

# validate UUIDs
uuid_validate(uuid_generate_random(2))
uuid_validate(uuid_generate_nil(2))
uuid_validate(uuid_generate_name(c("one", "two")))
uuid_validate(c("a", ""))
