parameters_dead R Documentation

Parameters data frame for dead marker measurements.

Description

Data frame containing the parameters calculated with processPeak function for dead marker measurements.

Usage

data("parameters_dead")

Format

A data frame with 13 observations on the following 9 variables.

compound

a factor with levels Kbr

flow

a numeric vector

tr

a numeric vector

Hmax

a numeric vector

A60

a numeric vector

B60

a numeric vector

A10

a numeric vector

B10

a numeric vector

area

a numeric vector
