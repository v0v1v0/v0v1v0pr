job_compliance R Documentation

Job compliance

Description

Job compliance

Usage

job_compliance(ids, name, type, resumable = TRUE, token = NULL)

Arguments

ids

Ids of tweets or users to check.
name

Name of the job.
type

Type of ids: either 'tweets' or 'users'.
resumable

A logical value if the job is resumable.
token

Bearer token used.
