amZoomButtons R Documentation

Zoom buttons

Description

Zoom buttons.

Usage

amZoomButtons(
  halign = "left",
  valign = "top",
  marginH = 5,
  marginV = 5,
  zoomFactor = 0.1
)

Arguments

halign

"left" or "right"
valign

"top" or "bottom"
marginH

horizontal margin
marginV

vertical margin
zoomFactor

zoom factor

Value

A list of parameters for zoom buttons, for usage in amLineChart or amScatterChart
