set.as R Documentation

RHMS objects connector

Description

this function connects a base object as a either of: 'downstream' or 'divertTo' to a target object, which are both instantiated by RHMS constructors.

Usage

set.as(base,target,type='downstream')

Arguments

base

An object; from either of classes of createReservoir, createJunction, createDiversion, createSubbasin, or createReach
target

An object; from either of classes of createReservoir, createJunction, createDiversion, createSubbasin, or createReach
type

the type of base object to be set as to the target object: 'downstream', or 'divertTo'

Value

an object from class of target object.
