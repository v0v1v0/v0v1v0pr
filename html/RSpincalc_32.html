Qinv R Documentation

Quaternion inverse

Description

Qinv calculated the quaternion inverse.

Usage

Qinv(Q)

Arguments

Q

Quaternion (Q) vector [q1, q2, q3, q4].

Value

Q

Quaternion inverse (Q) vector [q1, q2, q3, q4].

Author(s)

Jose Gama

Examples

Q <- c(-0.1677489, -0.7369231, -0.3682588, 0.5414703)
Qinv(Q)
