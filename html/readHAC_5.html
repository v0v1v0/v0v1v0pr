Extract tuples.

Description

Extract tuples of HAC object.

Usage

## S3 method for class 'HAC'
x[i, ...]

Arguments

x

HAC object
i

Integer vector
...

Currently not used

Details

Extract subset of tuples. For instance x[1:2] extracts the first two tuples. Alternatively the method can be indirectly invoked by the subset function.

Value

HAC object

Examples


x[1:2]
subset(x, type == 10000)
split(x, x$type)
