countries R Documentation

Countries

Description

Country codes, names, UN major areas and UNHCR regional bureaux/operations.

Usage

countries

Format

A tibble.

iso_code

ISO3 Country code.

unhcr_code

UNHCR Country code.

name

UNSD Name.

unhcr_region

UNHCR Region name.

unsd_region

UNSD Region name.

unsd_subregion

UNSD Sub-region name.

unsd_imregion

SDG Region name.

sdg_region

SDG Region name.

Source

UNHCR Refugee Data Finder.

Examples

countries
