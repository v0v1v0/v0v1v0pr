logout R Documentation

Logout of RobinHood

Description

Send a logout call through the RobinHood API service and disable your token.

Usage

logout(RH)

Arguments

RH

object of class RobinHood

Examples

## Not run: 
# Login in to your RobinHood account
RH <- RobinHood("username", "password")

logout(RH)

## End(Not run)
