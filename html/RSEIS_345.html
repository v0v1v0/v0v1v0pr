xprod R Documentation

Vector Cross Product

Description

Cross product of two vectors

Usage

xprod(A1, A2)

Arguments

A1

3 component vector of x,y,z
A2

3 component vector of x,y,z

Value

3 component vector of A1 cross A2

Author(s)

Jonathan M. Lees <jonathan.lees@unc.edu>

Examples

B1 <- c(4,9,2)
B2 <- c(2,-5,4)

xprod(B1, B2)
