repo_stashclear R Documentation

Remove all stashed data

Description

Remove all stashed data

Usage

repo_stashclear(force = F)

Arguments

force

If TRUE, no confirmation is asked.

Value

Used for side effects.

See Also

repo_rm, repo_stash

Examples

## Not run: 
rp_path <- file.path(tempdir(), "example_repo")
rp <- repo_open(rp_path, TRUE)
tempdata <- runif(10)
rp$stash("tempdata")
rp$print(all=TRUE)
rp$stashclear(TRUE)

## wiping temporary repo
unlink(rp_path, TRUE)

## End(Not run)
