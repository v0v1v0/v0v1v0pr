getError

Description

Retrieve a http or XML parsing error from an eutil object.

Usage

getError(x, ...)

## S4 method for signature 'eutil'
getError(x, ...)

Arguments

x

An eutil object.
...

Further arguments passed on to methods.

Value

An eutil_error object.

Methods (by class)
