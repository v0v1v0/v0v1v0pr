row_paths R Documentation

Get a list of table row/column paths

Description

Get a list of table row/column paths

Usage

row_paths(x)

col_paths(x)

Arguments

x

(VTableTree)
an rtable object.

Value

A list of paths to each row/column within x.

See Also

cell_values(), fnotes_at_path<-, row_paths_summary(), col_paths_summary()

Examples

lyt <- basic_table() %>%
  split_cols_by("ARM") %>%
  analyze(c("SEX", "AGE"))

tbl <- build_table(lyt, ex_adsl)
tbl

row_paths(tbl)
col_paths(tbl)

cell_values(tbl, c("AGE", "Mean"), c("ARM", "B: Placebo"))
