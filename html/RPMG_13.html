getmem R Documentation

Get Member

Description

Get a member of a list

Usage

getmem(v, mem = 1)

Arguments

v

vector
mem

element in vector

Details

Used in conjunction with apply

Value

vector of members of a list

Author(s)

Jonathan M. Lees<jonathan.lees@unc.edu>

Examples


z = list()
for(i in 1:10)
{

z[[i]] = round(10*runif(10))


}

y = as.vector(unlist(lapply(z, getmem, 6)))
