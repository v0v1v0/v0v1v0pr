Calculates the kurtosis of a Johnson Quantile-Parameterised Distribution.

Description

Calculates the kurtosis of a Johnson Quantile-Parameterised Distribution.

Usage

jqpd_kurtosis(params)

Arguments

params

jqpd object created using jqpd()

Value

The kurtosis of the distribution as a length one numeric vector.
