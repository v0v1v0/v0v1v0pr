tg.sampleProblem

Description

Returns sample TextGrid with continuity problem.

Usage

tg.sampleProblem()

Value

TextGrid

See Also

tg.repairContinuity

Examples

tg <- tg.sampleProblem()
tg2 <- tg.repairContinuity(tg)
tg2 <- tg.repairContinuity(tg2)
