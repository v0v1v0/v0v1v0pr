unuran.distr-class R Documentation

Virtual class "unuran.distr"

Description

The virtual class unuran.distr provides an interface to UNU.RAN objects for distributions.

The following classes extend this class:

class unuran.cont

for univariate continuous distributions, see unuran.cont.

class unuran.discr

for discrete distributions, see unuran.discr.

class unuran.cmv

for multivariate continuous distributions, see unuran.cmv.
