writeRaw.Verbose R Documentation

Writes objects if above threshold

Description

Writes objects if above threshold. This method is used by all other methods of this class for output.

Usage

## S3 method for class 'Verbose'
writeRaw(this, ..., sep="", level=this$defaultLevel)

Arguments

...

Objects to be passed to paste().
sep

The default separator character string.
level

A numeric value to be compared to the threshold.

Value

Returns nothing.
