make_gadget R Documentation

Make Gadget

Description

Create gadget for generating R6 methods. Action after clicking 'Done' button is defined by 'done_fun'.

Usage

make_gadget(title, title_bar, done_fun)

Arguments

title

Character, title of gadget window
title_bar

Character, gadget title bar
done_fun

Function to be used after clicking 'Done' button

Value

Function creating and running a Shiny gadget
