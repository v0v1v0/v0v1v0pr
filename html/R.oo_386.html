Description

Checks if a field exists or not.

Usage

## S3 method for class 'BasicObject'
hasField(this, field, ...)

Arguments

field

vector of fields to be checked if they exists or not.
...

Not used.

Value

Returns a logical vector indicating for each field if it exists or not.
