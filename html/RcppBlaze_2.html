Description

Set/Get the Number of Threads used in blaze

Usage

blaze_set_num_threads(n)

blaze_get_num_threads()

Arguments

n

The number of threads to set in blaze.

Value

blaze_get_threads returns an integer and blaze_set_threads returns nothing.
