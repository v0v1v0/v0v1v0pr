col R Documentation

Parameters data frame for columnar measurements.

Description

Data frame containing the parameters calculated with processPeak function for columnar measurements of several compounds.

Usage

data("col")

Format

A data frame with 50 observations on the following 9 variables.

compound

a factor with levels sulfi cloro dimeti Mera

flow

a numeric vector

tr

a numeric vector

Hmax

a numeric vector

A60

a numeric vector

B60

a numeric vector

A10

a numeric vector

B10

a numeric vector

area

a numeric vector
