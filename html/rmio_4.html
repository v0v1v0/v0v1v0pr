file_create R Documentation

Create or resize files

Description

Create or resize files

Usage

file_create(file, size)

file_resize(file, size)

file_resize_off(file, size)

Arguments

file

Path name.
size

Size in bytes.

Value

Input file, invisibly.

Examples

tmp <- tempfile()
file_create(tmp, 10)
file.size(tmp)
file_resize(tmp, 5)
file.size(tmp)
file_resize_off(tmp, 10)
file.size(tmp)
