setValue.ProgressBar R Documentation

Sets current value

Description

Sets current value. Note that this method does not update the bar visually.

Usage

## S3 method for class 'ProgressBar'
setValue(this, value, ...)

Arguments

value

A numeric in [0,maxValue].
...

Not used.

Value

Returns old value.
