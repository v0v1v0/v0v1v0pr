Quaternion dot product

Description

%Q.% performs a quaternion dot product.

Usage

Q1 %Q.% Q2

Arguments

Q1

Quaternion (Q) vector [q1, q2, q3, q4].

Q2

Quaternion (Q) vector [q1, q2, q3, q4].

Value

Q

Quaternion result of dot product (Q) vector [q1, q2, q3, q4].

Author(s)

Jose Gama

Examples

## Not run: 
Q <- c(-0.1677489, -0.7369231, -0.3682588, 0.5414703)
Q %Q.% Q

## End(Not run)
