cran_stats_by_package R Documentation

CRAN Stats for single package

Description

Download CRAN data related to several packages in a range of dates

Usage

cran_stats_by_package(from_date, to_date, sel_package_name,
  cran_log = "http://cran-logs.rstudio.com/")

Arguments

from_date

Starting date for downloading CRAN Repositories
to_date

end date for downloading CRAN Repositories
sel_package_name

Single package name for analysis
cran_log

CRAN mirror name
