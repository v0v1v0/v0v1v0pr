rseedcalc-package R Documentation

Estimation of the proportion of genetically modified stacked seeds in seedlots

Description

Estimate the percentage of seeds in a seedlot that contain stacks of genetically modified traits.

Details

The main functions for this package are stack2 and stack3.

For a complete list of functions, use library(help="rseedcalc")
