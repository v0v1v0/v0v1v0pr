ggacf R Documentation

Plot the Acf Figure

Description

Plot the Acf figure with observations of a single variable beautifully.

Usage

ggacf(data, lag = 10)

Arguments

data

vector of observations
lag

the maximum lag to calculate the acf

Value

A ggplot figure of the acf

Examples

swan <- rGarch(len = 180)
ggacf(swan^2, 20)
