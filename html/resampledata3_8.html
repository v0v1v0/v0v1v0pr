Cereals R Documentation

Cereals

Description

Nutritional data on a sample of cereals.

Usage

Cereals

Format

A data frame with 43 observations on the following 5 variables.

ID

a numeric vector

Age

target consumer, adult or children

Shelf

location of cereal, bottom, middle, or top shelf

Sodiumgram

sodium content in grams

Proteingram

protein content in grams
