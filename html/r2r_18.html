hash_fn R Documentation

Get hash function of an hash table

Description

Returns the hash function used for key hashing in an hash table (hashset or hashmap).

Usage

hash_fn(x)

## S3 method for class 'r2r_hashtable'
hash_fn(x)

Arguments

x

an hashset or hashmap.

Value

a function.

Author(s)

Valerio Gherardi

Examples

s <- hashset()
hash_fn(s)
