fact_table R Documentation

fact_table S3 class

Description

A fact_table object is created, we have to get its surrogate keys.

Usage

fact_table(
  name = NULL,
  surrogate_keys = NULL,
  agg = NULL,
  dim_int_names = NULL,
  instances = NULL
)

Arguments

name

A string, fact name.
surrogate_keys

A vector of strings, surrogate key names.
agg

A vector of strings, aggregation functions.
dim_int_names

A vector of strings, internal names of dimensions.
instances

A flat table with the fact instances.

Value

A fact_table object.
