isInt R Documentation

isInt

Description

Returns TRUE / FALSE whether it is exactly 1 integer number (in fact, the class can be numeric but the number must be integer), non-missing

Usage

isInt(num)

Arguments

num

variable to be tested

Value

TRUE / FALSE

See Also

isNum, isLogical, isString

Examples

isInt(2)
isInt(2L)
isInt(-2)
isInt(-2L)
isInt(2.1)
isInt(-2.1)
isInt(1:5)
isInt(NA_integer_)
isInt(integer(0))
