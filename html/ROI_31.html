Replicate "==", ">=" and "<=" Signs

Description

The utility functions eq, leq and geq replicate the signs "==", ">=" and "<=" n times.

Usage

eq(n)

leq(n)

geq(n)

Arguments

n

an integer giving the number of times the sign should be repeated.

Examples

eq(3)
leq(2)
geq(4)
