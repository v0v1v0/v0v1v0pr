print_leaderboard R Documentation

Format the leaderboard in Markdown.

Description

Format the leaderboard in Markdown.

Usage

print_leaderboard(
  best,
  metrics = names(metrics),
  test_name = "quiz",
  digits = 3,
  ...
)

Arguments

best

list of the best submissions per team and per metric as returned by get_best.
metrics

character vector. names of the metrics to be displayed
test_name

string. name of the test set used: "quiz" or "test"
digits

integer. how many significant digits are to be used for metrics.
...

further parameters to pass to kable

Value

print_leaderboard returns a character vector of the table source code to be used in a Markdown document.

Note

Chunk option results='asis' has to be used
