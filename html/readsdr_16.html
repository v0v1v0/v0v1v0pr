sd_interpret_estimates R Documentation

Interpret estimates

Description

Interpret estimates

Usage

sd_interpret_estimates(estimates, par_list)

Arguments

estimates

A list or data frame
par_list

A list

Value

A data frame

Examples

  estimates <- c(par_beta = 0,
                 par_rho  = 0.8472979,
                 I0       = 0,
                 inv_phi  = -2.302585)

  par_list <- list(list(par_name  = "par_beta",
                        par_trans = "exp"),
                   list(par_name  = "par_rho",
                        par_trans = "expit"),
                   list(par_name  = "I0",
                        par_trans = "exp"),
                   list(par_name  = "phi",
                        par_trans = c("exp", "inv")))
  sd_interpret_estimates(estimates, par_list)
