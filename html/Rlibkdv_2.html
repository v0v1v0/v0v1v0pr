kdv R Documentation

Use KDV

Description

Efficient and accurate kernel density visualization.

Usage

kdv(
  longitude,
  latitude,
  bandwidth_s = 1000,
  row_pixels = 800,
  col_pixels = 640
)

Arguments

longitude

features' longitude
latitude

features' latitude
bandwidth_s

spatial bandwidth
row_pixels

row pixels
col_pixels

col pixels

Value

kdv result

Examples


data(hk)
resKDV <- kdv(hk$lon, hk$lat, 1000, 800 ,640)
