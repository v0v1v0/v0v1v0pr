inlineDataFrame R Documentation

Inline dataframe

Description

Convert a string in dataframe and save it in a temporary file

Usage

inlineDataFrame(str)

Arguments

str

(string) Dataframe in string format

Value

dataframe object

Examples

## Not run: 
occ <- inlineDataFrame("
 id time occ
 1   0    1
 1   12   2
 1   24   3
 2   0    1
 2   24   2
 3   0    1
")

## End(Not run)
