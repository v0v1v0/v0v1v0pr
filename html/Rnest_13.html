pa R Documentation

Parallel analysis

Description

Parallel analysis

Usage

pa(
  data = NULL,
  n = NULL,
  p = NULL,
  nrep = 1000,
  alpha = 0.05,
  crit = NULL,
  ...
)

Arguments

data

data.frame.
n

number of subjects.
p

number of variables.
nrep

number of replications.
alpha

type I error rate.
crit

Critical values to compare the eigenvalues.
...

Other arguments

Value

nfactors (if data is supplied) and sampled eigenvalues

Examples

pa(ex_2factors, n = 42)
E <- pa(n = 10, p = 2, nrep = 5)
