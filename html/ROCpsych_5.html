ROC.data.ex R Documentation

Example data

Description

This hypothetical dataset contains records of the outcome, the predictor, gender, and age from 241 participants.

Usage

data("ROC.data.ex")

Format

A data frame with 241 observations on the following 4 variables.

outcome

a numeric vector

predictor

a numeric vector

gender

a numeric vector

age

a numeric vector

Examples
