mpower R Documentation

Power of a Matrix

Description

%^% calculates x^p for the square matrix, x, by spectral decomposition.

Usage

x%^%p

Arguments

x

A square matrix.
p

The power to which the matrix is to be raised.

Value

%^% returns the power of a matrix.

Author(s)

J.K. Lindsey

Examples

## Not run: 
x <- matrix(c(0.4,0.6,0.6,0.4),nrow=2)
x%^%2
x%^%10
x%^%20

## End(Not run)
