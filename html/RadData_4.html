rad_codes R Documentation

Radiation code table

Description

A small table providing association of code_num, code_AN and description

Usage

rad_codes

Format

A tibble with 3 variables.

A table used for cross-linking numerical, alphabetical and textual references for a radiation emission type, such as: 1, G, Gamma Rays.

code_num Numeric code for radiation type.

code_AN Alphabetical abbreviation, some with + or -.

description Description of radiation emission type.
