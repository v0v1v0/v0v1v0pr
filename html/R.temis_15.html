set_corpus_variables R Documentation

set_corpus_variables

Description

Set corpus meta-data variables from a data frame.

Usage

set_corpus_variables(corpus, dset)

Arguments

corpus

A Corpus object.
dset

A data.frame containing meta-data variables, with one row per document in 'corpus.

Value

A Corpus object with meta-data added.

Examples


file <- system.file("texts", "reut21578-factiva.xml", package="tm.plugin.factiva")
corpus <- import_corpus(file, "factiva", language="en")
dtm <- build_dtm(corpus)
dset <- data.frame(x=1:length(corpus))
corpus <- set_corpus_variables(corpus, dset)
