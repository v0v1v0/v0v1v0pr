Description

Make R code slide

Usage

Rcode2office(
  code,
  preprocessing = "",
  title = "",
  type = "pptx",
  target = "Report",
  append = FALSE
)

Arguments

code

A character string encoding R codes
preprocessing

A character string of R code as a preprocessing
title

A character
type

desired format. choices are "pptx" or "docx"
target

name of output file
append

logical
