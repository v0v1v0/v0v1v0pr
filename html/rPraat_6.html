as.tg R Documentation

as.tg

Description

Renames the class(tg)["name"] attribute and sets class(tg)["type"] <- "TextGrid" (if it is not already set)

Usage

as.tg(tg, name = "")

Arguments

tg

TextGrid object
name

New name

Value

TextGrid object

Examples

class(tg.sample())
class(as.tg(tg.sample(), name = "New Name"))
