ROI_plugin_get_solver_name R Documentation

Get Solver Name

Description

Get the name of the solver plugin.

Usage

ROI_plugin_get_solver_name(pkgname)

Arguments

pkgname

a string giving the package name.

Value

Returns the name of the solver as character.
