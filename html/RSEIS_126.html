INSTFREQS R Documentation

Instrument Frequencies

Description

Vector of frequencies

Usage

INSTFREQS(b, a, w)

Arguments

b

numerator, zeros
a

denominator, poles
w

frequency

Details

h = jpolyval(b,s) / jpolyval(a,s)

Value

h = jpolyval(b,s) / jpolyval(a,s)

Author(s)

Jonathan M. Lees<jonathan.lees.edu>

Examples


K  <-  PreSet.Instr()
b  <-  K[[1]]$zeros
a  <-  K[[1]]$poles
INSTFREQS(b, a, 2*pi*12)
