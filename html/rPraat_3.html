as.pitch R Documentation

as.pitch

Description

Renames the class(pitch)["name"] attribute and sets class(pitch)["type"] <- "Pitch 1" (if it is not already set)

Usage

as.pitch(pitch, name = "")

Arguments

pitch

Pitch 1 object
name

New name

Value

Pitch 1 object

Examples

class(pitch.sample())
class(as.pitch(pitch.sample(), name = "New Name"))
