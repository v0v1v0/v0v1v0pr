printWarnings.Verbose R Documentation

Outputs any warnings recorded

Description

Outputs any warnings recorded. The output is indented according to *enter()/*exit() calls.

Usage

## S3 method for class 'Verbose'
printWarnings(this, title="Warnings detected:", ..., level=this$defaultLevel)

Arguments

title

A character string to be outputted before the warnings, if they exists.
...

Arguments passed to *cat().
level

A numeric value to be compared to the threshold.

Value

Returns nothing.
