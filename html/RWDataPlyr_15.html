rdf_slot_names R Documentation

Returns all slots contained in an rdf file.

Description

rdf_slot_names() returns a character vector of all slots contained within an rdf object.

Usage

rdf_slot_names(rdf)

getSlotsInRdf(rdf)

Arguments

rdf

An rdf object.

Value

A character vector.
