getpeerinfo R Documentation

RPC-JSON API: getpeerinfo

Description

Returns data about each connected network node as a json array of objects.

Usage

getpeerinfo(con)

Arguments

con

object of class CONRPC.

Value

A S4-object of class ANSRPC.

Author(s)

Bernhard Pfaff
