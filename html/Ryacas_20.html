Matrix Power

Usage

pow(x, n, ...)

## Default S3 method:
pow(x, n, ...)

Arguments

x

If yac_symbol treat as such, else call pow.default().
n

nth power of the square matrix.
...

further arguments passed to pow.default()

Examples

(x <- matrix(c(1, 2, 2, 3), ncol = 2))
pow(x, 2)
pow(ysym(x), 2)
