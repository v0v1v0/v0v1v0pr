intRangeDay R Documentation

Integer range within a day

Description

This function returns the associated integer times for the start and end of a specific day.

Usage

intRangeDay(x)

Arguments

x

POSIXct, date/time object.

Value

integer

Author(s)

Bernhard Pfaff

Examples

d1 <- "2017-03-15"
intRangeDay(d1)
intMinDay(d1)
intMaxDay(d1)
