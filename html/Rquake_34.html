lastPIX R Documentation

Last Pix

Description

'RSEIS' Button: Restore Last WPX file from memory. Function is used internally in swig.

Usage

lastPIX(nh, g)
editPIX(nh, g)

Arguments

nh

GH list from RSEIS

g

parameters from swig

Value

New WPX list attached to g
