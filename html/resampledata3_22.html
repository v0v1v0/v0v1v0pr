Illiteracy R Documentation

Illiteracy

Description

Data on female illiteracy in a sample of countries where illiteracy is more than 5%.

Usage

Illiteracy

Format

A data frame with 94 observations on the following 4 variables.

ID

Country ID

Country

Name of country

Illit

Percentage of women over 15 years old who are illiterate (2003)

Births

Number of births per woman in that country (2005)

Source

www.unesco.org, www.data.worldbank.org
