dfp_getLabelsByStatement R Documentation

getLabelsByStatement

Description

Gets a LabelPage of Label objects that satisfy the given Statement query. The following fields are supported for filtering:

  • id

  • type

  • name

  • description

  • isActive

Usage

dfp_getLabelsByStatement(request_data, as_df = TRUE, verbose = FALSE)

Arguments

request_data

a list or data.frame of data elements to be formatted for a SOAP request (XML format, but passed as character string)
as_df

a boolean indicating whether to attempt to parse the result into a data.frame
verbose

a boolean indicating whether to print the service URL and POSTed XML

Value

a data.frame or list containing all the elements of a getLabelsByStatementResponse

Examples

## Not run: 
 dat <- list(filterStatement=list('query'="WHERE status='ACTIVE'")) 
 res <- dfp_getLabelsByStatement(dat)
