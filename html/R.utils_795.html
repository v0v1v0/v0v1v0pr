str.Verbose R Documentation

Prints the structure of an object if above threshold

Description

Prints the structure of an object if above threshold. The output is not indented.

Usage

## S3 method for class 'Verbose'
str(object, ..., level=this$defaultLevel)

Arguments

...

Objects to be passed to str.
level

A numeric value to be compared to the threshold.

Value

Returns nothing.
