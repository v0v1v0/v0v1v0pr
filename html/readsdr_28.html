sd_stocks R Documentation

Summarise the information of a model's stocks in a data frame

Description

Summarise the information of a model's stocks in a data frame

Usage

sd_stocks(mdl)

Arguments

mdl

A list which is the output from read_xmile.

Value

A data frame.

Examples

path <- system.file("models", "SIR.stmx", package = "readsdr")
mdl  <- read_xmile(path)
sd_stocks(mdl)
