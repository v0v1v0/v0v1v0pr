UTM.functions R Documentation

UTM Convenience Functions

Description

Functions for converting latitude-longitude data to UTM.

Usage

long2UTM(long)

UTMzones(long)

chooseUTM(long)

Arguments

long

A vector of longitude values.

Value

UTM vector of zone numbers.

UTM vector of zone numbers.

UTM zone number.

Examples

long2UTM(c(-90, 0, 90))
UTMzones(c(-90, 90, 90))
chooseUTM(c(-90, -80, -70))
