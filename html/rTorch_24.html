dim.torch.Tensor R Documentation

Dimensions of a tensor

Description

Get the dimensions of a tensor displaying it as a vector.

Usage

## S3 method for class 'torch.Tensor'
dim(x)

Arguments

x

tensor

Value

a vector of integers with the dimensions of the tensor

Examples
