AIC3 R Documentation

AIC with 3 penalty for randomLCA object

Description

Returns AIC with penalty 3 for a randomLCA object.

Usage

AIC3(object)

Arguments

object

randomLCA object

Value

AIC3.

Author(s)

Ken Beath

Examples


pap.lca2 <- randomLCA(pap[, 1:10], freq = pap$Freq, nclass = 2, cores = 1)
print(AIC3(pap.lca2))
