hash256 R Documentation

BTC hash256

Description

This function returns the hash by applying the sha256 hashing algorithm twice to a raw object.

Usage

hash256(d)

Arguments

d

raw, vector.

Value

character, the value of d hashed twice.

Author(s)

Bernhard Pfaff
