nrow,VTableTree-method R Documentation

Table dimensions

Description

Table dimensions

Usage

## S4 method for signature 'VTableTree'
nrow(x)

## S4 method for signature 'VTableNodeInfo'
ncol(x)

## S4 method for signature 'VTableNodeInfo'
dim(x)

Arguments

x

(TableTree or ElementaryTable)
a table object.

Value

The number of rows (nrow), columns (ncol), or both (dim) of the object.

Examples

lyt <- basic_table() %>%
  split_cols_by("ARM") %>%
  analyze(c("SEX", "AGE"))

tbl <- build_table(lyt, ex_adsl)

dim(tbl)
nrow(tbl)
ncol(tbl)

NROW(tbl)
NCOL(tbl)
