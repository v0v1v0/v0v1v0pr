deg2rad R Documentation

Convert degrees to radians

Description

Convert degrees to radians

Usage

deg2rad(deg)

Arguments

deg

vector of degrees to be converted to radians

Examples

data(vdendool)

## latitude in degrees
degrees <- coordinates(vdendool)[, 2]
head(degrees)

## latitude in radians
radians <- deg2rad(coordinates(vdendool)[, 2])
head(radians)
