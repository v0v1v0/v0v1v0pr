quadlength R Documentation

Find length and dyadic length of square matrix

Description

h(t) = (-1)^(t-1) * x(t), 1 <= t <= length(x)

Usage

quadlength(x)

Arguments

x

2-d image; dim(n,n), n = 2^J (hopefully).

Value

n length(x).

J least power of two greater than n.

Examples

quadlength(matrix(1:16,ncol=4))
