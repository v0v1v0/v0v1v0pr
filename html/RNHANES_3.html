demography_filename R Documentation

Translates cycle years into the correct demography filename suffix, e.g. '2001-2002' returns 'B'

Description

Translates cycle years into the correct demography filename suffix, e.g. '2001-2002' returns 'B'

Usage

demography_filename(year)

Arguments

year

NHANES cycle, e.g. "2001-2002"

Value

suffix character e.g. "B"
