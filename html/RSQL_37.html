sql_gen_update

Description

Generates an update statement

Usage

sql_gen_update(
  table,
  update_fields = names(values),
  values,
  where_fields = names(where_values),
  where_values
)

Arguments

table

The table to update
update_fields

The fields to update
values

The values to update
where_fields

The fields for where statement
where_values

The values for where statement
