clean_merge_params R Documentation

Description

Internal function for tidiness.

Usage

clean_merge_params(recolorize_obj, merge_list, color_to)

Arguments

recolorize_obj

Object of recolorize class.
merge_list

List of layers to merge.
color_to

Argument for coloring new layers.

Value
