DAYSperYEAR R Documentation

Days per Year

Description

Calculate the number of days per calendar year

Usage

DAYSperYEAR(yr)

Arguments

yr

year

Value

days

integer number of days for a given year

Author(s)

Jonathan M. Lees<jonathan.lees@unc.edu>

Examples


years <- seq(from=1850, to=2010, by=1)

DAYSperYEAR(years)
