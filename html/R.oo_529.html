getDevelUrl.Package R Documentation

Gets the URL(s) from where the developers version of this package can be installed

Description

Gets the URL(s) from where the developers version of this package can be installed by looking for comma or semicolon separated URLs at the optional DevelURL line in the DESCRIPTION file of the package.

Usage

## S3 method for class 'Package'
getDevelUrl(this, ...)

Arguments

...

Not used.

Value

Returns a vector of character strings.

Author(s)

Henrik Bengtsson

See Also

For more information see Package.

Examples

  pkg <- Package("R.oo")
  print(getDevelUrl(pkg))
