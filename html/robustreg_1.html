fit_rcpp R Documentation

Predict y from X and b

Description

Predict y vector from X design matrix and b vector

Usage

fit_rcpp(X,b)

Arguments

X

Design matrix
b

Estimates of beta

Author(s)

Ian M. Johnson

Examples

j <- rep(1, 5)
x1 <- rnorm(5)
x2 <- rnorm(5, 10, 20)
X = as.matrix(data.frame(j, x1, x2))
b <- 1:3
fit_rcpp(X, b)
