addtix R Documentation

add tix to plot

Description

Add tick marks to edge of plot

Usage

addtix(side = 3, pos = 0, tck = 0.005, at =c(0, 1), labels = FALSE, col = 2, ...)

Arguments

side

side of plot 1-4
pos

position relative to side
tck

tick size
at

locations along axis
labels

labels for tics
col

color for ticks
...

graphical parameters, par
