invsqrt R Documentation

Return inverse square root of square positive-definite matrix.

Description

Return inverse square root of square positive-definite matrix.

Usage

invsqrt(A)

Arguments

A

a numeric matrix

Value

inverse square root of square positive-definite matrix.

Examples

require(Rsfar)
X <- Bdiag(matrix(1:4,2,2), 3)
invsqrt(t(X) %*% X)
