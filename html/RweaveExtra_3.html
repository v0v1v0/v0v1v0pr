RweaveExtraLatexUtils R Documentation

RweaveExtra Driver Utilities

Description

These functions are considered internal and hence not documented. The first mostly calls makeRweaveLatexCodeRunner, whereas the second is a slightly modified version of RtangleRuncode, both from package utils.

Usage

RweaveExtraLatexRuncode(object, chunk, options)
RtangleExtraRuncode(object, chunk, options)

Value

No return value; called for their side effects.

Author(s)

Vincent Goulet, based on work by Friedrich Leisch, R-core and Seth Falcon.

See Also

Sweave, RweaveLatex, Rtangle.
