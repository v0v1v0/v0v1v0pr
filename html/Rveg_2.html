addReleve

Description

Writing and editing your releves

Usage

addReleve(
  DATABASE = "NEW",
  SAVE = "default",
  checklist = "default",
  extrahead = NULL
)

Arguments

DATABASE

name of csv files for releve table and header - database
SAVE

name of exporting database
checklist

custom checklist
extrahead

extra rows in header

Value

export two csv files, one for releve and one for header
