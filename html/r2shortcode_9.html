isUpperCase R Documentation

Is string uppercase?

Description

Simply, test if a string is uppercase and return TRUE or FALSE

Usage

isUpperCase(string)

Arguments

string

The string to evaluate

Value

TRUE or FALSE if the string is all upper case

Examples

strTest1 <- 'OBI'
strTest2 <- 'obiO'
isUpperCase(strTest1)
isUpperCase(strTest2)
