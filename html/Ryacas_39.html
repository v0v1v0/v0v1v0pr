yac R Documentation

Run yacas command

Description

Run yacas command

Usage

yac(x, rettype = c("str", "expr", "silent"))

Arguments

x

yacas command
rettype

str for string/character, expr for expression, silent for silent

Examples

yac("D(x) x^2 + 4*x")
yac("D(x) x^2 + 4*x", rettype = "str")
yac("D(x) x^2 + 4*x", rettype = "expr")
yac("D(x) x^2 + 4*x", rettype = "silent")
