Mmorlet R Documentation

Morlet Wavelet

Description

Make Morlet Wavelet

Usage

Mmorlet(UB = -4, LB = 4, N = 256, plot = FALSE)

Arguments

UB

upper bound

LB

lower bound
N

number of points
plot

logical, TRUE=plot

Details

create a morlet function based on the matlab style routines

Value

time series list:

xval

x-output
morl

y-output

Author(s)

Jonathan M. Lees<jonathan.lees.edu>

See Also

scal2freqs, Rwave

Examples


mm  <- Mmorlet(-8, 8, 256)
