Description

Calculates sum of squared errors, divided by NT

Usage

Arguments

matrix with error terms
NN

N
TT

T

Value

numeric

Examples

Y <- Y_dgp3
set.seed(1)
e <- matrix(rnorm(nrow(Y) * ncol(Y)), nrow = nrow(Y))
calculate_sigma2(e)
