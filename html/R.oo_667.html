unload.Package R Documentation

Unloads a package

Description

Unloads a package. This is an alternative way to use detach() to unload a package. If the package is not loaded, it will quietly return.

Usage

## S3 method for class 'Package'
unload(this, ...)

Arguments

...

Not used.

Value

Returns nothing.

Author(s)

Henrik Bengtsson

See Also

*load(). *isLoaded(). search(). For more information see Package.

Examples

## Not run: 
pkg <- Package("boot")
load(pkg)
print(isLoaded(pkg))
unload(pkg)
print(isLoaded(pkg))


## End(Not run)
