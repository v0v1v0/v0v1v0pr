label.it R Documentation

Labels on Plots

Description

Put Labels (A,B, C...) on corners of figures

Usage

label.it(a = "", corn = 1, ...)

Arguments

a

letters
corn

corner
...

graphical parameters passed from par

Value

Graphical Side effects

Author(s)

Jonathan M. Lees<jonathan.lees@unc.edu>

Examples


par(mfrow=c(2,2))
for(i in 1:4)
{
plot(rnorm(5), rnorm(5))
label.it(letters[i],1)
}
