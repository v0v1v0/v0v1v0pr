Description

Add parentheses around standard error estimates

Usage

add_parens(x, digits = 3)

Arguments

x

Numeric vector
digits

Number of digits to retain

Value

A character vector with enclosing parentheses

Examples


std.error <- c(0.12, 0.001, 1.2)
add_parens(std.error)
