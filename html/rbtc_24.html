date2int R Documentation

Convert date/time to integer

Description

This function returns the associated integer time for a given date/time object (coercible as POSIXct object.

Usage

date2int(x)

Arguments

x

POSIXct, date/time object.

Value

integer

Author(s)

Bernhard Pfaff

Examples

d <- "2017-03-15"
date2int(d)
