Datasets: Tidal Harmonic Components

Description

Harmonic components for tides

Usage

data("VelocityTide")
data("EastportTide")
data("CutlerFarrisTide")
data("PortAransasTide")
data("AnchorageTide")

Format

Harmonic components

Details

Tidal calculations
