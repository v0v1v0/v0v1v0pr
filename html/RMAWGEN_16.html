extractTxFromAnomalies R Documentation

Extracts generated time series of Daily Maximum Temperature from a random multi-realization obtained by generateTemperatureTimeseries function

Description

Extracts generated time series of Daily Maximum Temperature from a random multi-realization obtained by generateTemperatureTimeseries function

Usage

extractTxFromAnomalies(res_multigen, std, SplineAdv)

Arguments

res_multigen

matrix containing standardized values of daily temperature as returned by generateTemperatureTimeseries (first item)
std

vector containing standard deviation for each maximum temperature anomalies
SplineAdv

matrix containing the averaged values of maximum temperature obtained by a spline interpolation of monthly climate

Value

a matrix with generated maximum temperature
