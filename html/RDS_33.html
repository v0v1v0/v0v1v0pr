get.wave R Documentation

Calculates the depth of the recruitment tree (i.e. the recruitment wave) at each node.

Description

Calculates the depth of the recruitment tree (i.e. the recruitment wave) at each node.

Usage

get.wave(data)

Arguments

data

An rds.data.frame

Examples

data(fauxmadrona)
#number subjects in each wave
w <- get.wave(fauxmadrona)
#number recruited in each wave
barplot(table(w))
