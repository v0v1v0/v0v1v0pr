Cafeteria R Documentation

Cafeteria

Description

Nutritional data on meals served in a college cafeteria.

Usage

Cafeteria

Format

A data frame with 41 observations on the following 9 variables.

ID

a numeric vector

Type

type of meal, Meat or Vegetarian

Calories

number of calories

Carbohydrates

number of carbohydrates

Fiber

fiber content

Fat

fat content

Cholesterol

cholesterol

Protein

protein

Sodium

sodium

Source

Stephenson (private communication).
