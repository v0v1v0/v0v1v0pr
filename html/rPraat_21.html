ifft R Documentation

ifft

Description

Inverse Fast Fourier Transform (discrete FT), Matlab-like behavior.

Usage

ifft(sig)

Arguments

sig

input vector

Details

This really is the inverse of the fft function, so ifft(fft(x)) == x.

Value

output vector of the same length as the input vector

See Also

fft, Re, Im, Mod, Conj

Examples

ifft(fft(1:5))
