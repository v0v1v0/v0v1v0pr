rorq_mandible R Documentation

Rorqual projected mandible length

Description

Rorqual projected mandible length

Usage

rorq_mandible(species, length_m)

Arguments

species

a vector of species codes
length_m

a vector of lengths in meters

Value

a vector of laterally projected mandible lengths in m

Examples

# A 22m blue whale
rorq_mandible("bw", 22)

# A 7m minke
rorq_mandible("ba", 7)
