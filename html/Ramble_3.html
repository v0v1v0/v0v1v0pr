Description

%then% is the infix operator for the then combinator.

Usage

p1 %then% p2

Arguments

p1

the first parser
p2

the second parser

Value

recognises anything that p1 and p2 would if placed in succession.

Examples

(item() %then% succeed("123")) ("abc")
