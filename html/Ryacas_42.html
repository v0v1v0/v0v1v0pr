yac_expr R Documentation

Run yacas command returning R expression

Description

Run yacas command returning R expression

Usage

yac_expr(x)

Arguments

x

yacas command

Examples

yac_expr("D(x) x^2 + 4*x")
yac_expr("Limit(x, 1) (x^2 - 1)/(x - 1)")
yac_expr("Sum(n, 1, Infinity, (1/2)^n)")
yac_expr("Fibonacci(10)")
yac_expr("Sum(n, 1, 10, Fibonacci(n))")
