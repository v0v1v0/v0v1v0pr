Qnorm R Documentation

Norm of a quaternion

Description

Qnorm calculates the norm of a quaternion.

Usage

Qnorm(Q)

Arguments

Q

Quaternion (Q) vector [q1, q2, q3, q4].

Value

Norm of the quaternion.

Author(s)

Jose Gama

Examples

Q <- c(-0.1677489, -0.7369231, -0.3682588, 0.5414703)
Qnorm(Q)
