tg.removePoint R Documentation

tg.removePoint

Description

Remove point of the given index from the point tier.

Usage

tg.removePoint(tg, tierInd, index)

Arguments

tg

TextGrid object
tierInd

tier index or "name"
index

index of point to be removed

Value

TextGrid object

See Also

tg.insertPoint, tg.getNumberOfPoints, tg.removeIntervalBothBoundaries

Examples

tg <- tg.sample()
tg$phoneme$label
tg2 <- tg.removePoint(tg, "phoneme", 1)
tg2$phoneme$label
