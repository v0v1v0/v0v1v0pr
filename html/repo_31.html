repo_root R Documentation

Show path to repo root

Description

Show path to repo root

Usage

repo_root()

Value

character containing the path to the root of the repo.

Examples

rp_path <- file.path(tempdir(), "example_repo")
rp <- repo_open(rp_path, TRUE)
print(rp$root())

## wiping temporary repo
unlink(rp_path, TRUE)
