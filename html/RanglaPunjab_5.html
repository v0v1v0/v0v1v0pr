RanglaPunjab R Documentation

Palette of 5 Colors

Description

This function returns a palette of 5 colors

Usage

RanglaPunjab(name = NULL, ...)

Arguments

name

Palette name
...

Numeric, complex, or logical vectors

Value

Vector of 5 color values

Examples

RanglaPunjab("GoldenTemple")
RanglaPunjab("SohniMahiwal")
RanglaPunjab("Teej")
