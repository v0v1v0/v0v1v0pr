Description

ERDDAP server URLS and other info

Usage

servers(...)

Arguments

...

curl options passed on to crul::verb-GET

Value

data.frame with 3 columns:

  • name (character): ERDDAP name

  • url (character): ERDDAP url

  • public (logical): whether it's public or not
