print.multisimsum

Description

Print method for multisimsum objects

Usage

## S3 method for class 'multisimsum'
print(x, ...)

Arguments

x

An object of class multisimsum.
...

Ignored.

Examples

data(frailty)
ms <- multisimsum(
  data = frailty, par = "par", true = c(
    trt = -0.50,
    fv = 0.75
  ), estvarname = "b", se = "se", methodvar = "model",
  by = "fv_dist"
)
ms
