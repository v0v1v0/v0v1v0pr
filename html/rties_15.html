indiv3profilesCat R Documentation

Produces plots for sysVarIn when sysVar is categorical and there are 3 profiles

Description

Produces plots for sysVarIn when sysVar is categorical and there are 3 profiles

Usage

indiv3profilesCat(basedata, testModel, sysVar0name, sysVar1name)

Arguments

basedata

A dataframe created internally by the "sysVarInPlots" function.
testModel

The model object created by sysVarIn for the interaction model (e.g., sysVarInteract)
sysVar0name

The name created by sysVarInPlots referring to the system variable for partner-0.
sysVar1name

The name created by sysVarInPlots referring to the system variable for partner-1.

Value

A list of 3 plots showing the simple slopes for each of the profiles.
