m_add_style R Documentation

Overwrite Previous Style

Description

Takes a selection and overwrites previous styling with given styles.

Usage

m_add_style(id, style = m_style_cartoon(), sel = m_sel())

Arguments

id

R3dmol id or a r3dmol object (the output from r3dmol())
style

Style spec to apply to specified atoms using m_style_*()
sel

Atom selection specification with m_sel()

Value

R3dmol id or a r3dmol object (the output from r3dmol())

Examples

library(r3dmol)

# Add style to model
r3dmol() %>%
  m_add_model(data = pdb_1j72, format = "pdb") %>%
  m_add_style(style = m_style_cartoon()) %>%
  m_zoom_to()

# Set style to model
r3dmol() %>%
  m_add_model(data = pdb_6zsl, format = "pdb") %>%
  m_set_style(style = m_style_cartoon()) %>%
  m_set_style(
    sel = m_sel(chain = "A"),
    style = m_style_stick(
      radius = 0.5,
      colorScheme = "magentaCarbon"
    )
  ) %>%
  m_zoom_to()
