Apply Hard Threshold

Description

Apply Hard Threshold

Usage

HardThresh(y, t)

Arguments

y

Noisy Data.
t

Threshold.

Value

x filtered result (y 1_|y|>t).

See Also

SoftThresh.

Examples

f <- MakeSignal('HeaviSine',2^3)
qmf <- MakeONFilter('Daubechies', 10)
L <- 0
wc <- FWT_PO(f, L, qmf)
thr <- 2
wct <- HardThresh(wc, thr)
fhard <- IWT_PO(wct, L, qmf)
