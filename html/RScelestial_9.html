Li R Documentation

Bladder invasive single cell tumor dataset

Description

Bladder invasive single cell tumor dataset

Usage

data(Li)

Format

Each column represent a cell and each row represent a locus. "./." represent the missing value, "A/A" the normal state and "C/C" the mutated state.
