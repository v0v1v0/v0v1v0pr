set_time_window R Documentation

Set time window

Description

Sets the time window to be used by R5.

Usage

set_time_window(r5r_core, time_window)

Arguments

r5r_core

An object to connect with the R5 routing engine, created with setup_r5().
time_window

A number.

Value

Invisibly returns TRUE.
