sort_triple_hclust R Documentation

Sort Triple Hclust

Description

Sort a triple_omic object using hierarchical clustering

Usage

sort_triple_hclust(triple_omic, sort_table, value_var)

Arguments

triple_omic

an object of class triple_omic produced by create_triple_omic
sort_table

samples or features
value_var

An abundance value to use with hclust

Value

A triple_omic with clustered features or samples.
