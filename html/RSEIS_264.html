secdifv R Documentation

Seconds Difference

Description

Given two date/time vectors, return seconds diffrence

Usage

secdifv(T1, T2)

Arguments

T1

c(jd, hr, mi, sec)
T2

c(jd, hr, mi, sec)

Details

Year is not included in this calculation.

Value

numeric

seconds

Author(s)

Jonathan M. Lees<jonathan.lees.edu>

See Also

secdif

Examples

T1 <- c(12, 13, 23, 21)
T2 <- c(14, 23, 23, 2)
secdifv(T1, T2)
