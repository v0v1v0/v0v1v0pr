fCalcSVPfromTair R Documentation

fCalcSVPfromTair

Description

Calculate SVP (of water) from Tair

Usage

fCalcSVPfromTair(Tair = Tair.V.n, Tair.V.n)

Arguments

Tair

Data vector of air temperature (Tair, degC)
Tair.V.n

deprecated

Value

Data vector of saturation vapor pressure (SVP, hPa (mbar))
